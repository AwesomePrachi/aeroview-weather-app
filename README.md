# 🌤️ AeroView — Live Weather App

AeroView is a modern, responsive weather web application that provides real-time weather data and 5-day forecasts using live API integration.

Built using **Vanilla JavaScript + TailwindCSS**, this project demonstrates API consumption, asynchronous data handling, geolocation support, and dynamic UI updates.

---

## 🚀 Live Demo

🔗 https://aeroview.vercel.app/

---

## 📌 Features

- 🌍 Search weather by city name  
- 📍 Auto-detect weather using geolocation  
- 📅 5-Day weather forecast  
- 🌡️ Temperature unit toggle (°C / °F)  
- ⛅ Dynamic weather-based background  
- 🎨 Modern responsive UI  
- 🔄 Loading states & error handling  
- 📡 Real-time data from OpenWeatherMap API

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/aeroview-weather-app.git
cd aeroview-weather-app
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Create .env file

Create a `.env` file in the root directory:

```env
VITE_WEATHER_API_KEY=your_openweathermap_api_key
```

### 4️⃣ Run the app

```bash
npm run dev
```

Open http://localhost:5173 in your browser.

---

## 📝 API Configuration

This project uses **OpenWeatherMap API**.

1. Get your free API key from [openweathermap.org](https://openweathermap.org/api)
2. Add it to `.env` as `VITE_WEATHER_API_KEY`
